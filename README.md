# AI Boyfriend Chatbot

A personality-based chatbot built with Large Language Models (LLMs).

This project uses Groq API with customized Prompt Engineering to create a role-playing conversational agent with consistent personality, conversation style, and memory capability.

## Features

### Persona-based Conversation

- Designed a customized chatbot personality using Prompt Engineering.
- Controls character behavior, speaking style, and interaction patterns.
- Maintains consistent role-play responses throughout conversations.

### Memory System

- Stores user information using JSON-based memory.
- Injects saved information into prompts to create personalized conversations.
- Improves continuity across multiple conversations.

### Context-aware Dialogue

- Maintains conversation context.
- Responds based on previous messages.
- Reduces repetitive and unnatural responses.

### Gradio Interface

- Provides a simple web-based chat interface.
- Supports real-time chatbot interaction.
